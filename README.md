# Простий калькулятор

Консольний калькулятор для виконання базових арифметичних операцій.
Навчальний проєкт для вивчення Git, GitHub та Python.

## Опис
Цей проєкт представляє собою простий калькулятор з текстовим інтерфейсом, який дозволяє виконувати основні математичні операції в інтерактивному режимі.

Детальний опис функцій може бути наведено у файлі `DOCUMENTATION.md` (якщо є).

## Функціональність

- Додавання двох чисел
- Віднімання двох чисел
- Множення двох чисел
- Ділення двох чисел (з перевіркою ділення на нуль)
- Піднесення до степеня
- Обчислення остачі від ділення (модуль)

## Таблиця функцій

| Функція | Операція | Приклад виклику |
| :--- | :--- | :--- |
| `add` | Додавання (+) | `add(2, 3) -> 5` |
| `subtract` | Віднімання (-) | `subtract(5, 2) -> 3` |
| `multiply` | Множення (*) | `multiply(4, 5) -> 20` |
| `divide` | Ділення (/) | `divide(10, 2) -> 5.0` |
| `power` | Степінь (**) | `power(2, 3) -> 8` |
| `modulo` | Остача (%) | `modulo(10, 3) -> 1` |

## Вимоги

- Python 3.7 або вище
- Операційна система: Windows, macOS, Linux

## Встановлення

1. Клонуйте репозиторій:

```powershell
git clone https://github.com/Denus-IT/calculator-project.git
```

2. Перейдіть у папку проєкту:

```powershell
cd calculator-project
```

3. (Опціонально) Створіть та активуйте віртуальне середовище (Windows PowerShell):

```powershell
python -m venv .venv
.\.venv\Scripts\Activate
```

4. Запустіть програму:

```powershell
python calculator.py
```

> Порада: на macOS/Linux замініть активацію віртуального середовища на `source .venv/bin/activate`.

## Використання

Після запуску програми:

1. Оберіть операцію: `+`, `-`, `*`, `/`, `**` або `%`.
2. Введіть перше число.
3. Введіть друге число.
4. Отримайте результат.
5. Для виходу введіть `exit`.

### Приклад роботи

```text
=== Простий калькулятор ===
Операції: +, -, *, /, **, %
Для виходу введіть 'exit'

Введіть операцію (+, -, *, /, **, %) або 'exit': +
Введіть перше число: 15
Введіть друге число: 7
Результат: 22.0

Введіть операцію (+, -, *, /, **, %) або 'exit': **
Введіть перше число: 2
Введіть друге число: 8
Результат: 256.0
```
